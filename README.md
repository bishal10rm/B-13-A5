1.What is the difference between var, let, and const?

var, let, and const are used to declare variables in JavaScript.var is function scoped. It can be redeclared and its value can be changed.let is block scoped. It cannot be redeclared, but its value can be changed.
const is also block scoped. It cannot be redeclared and its value cannot be changed.

2.What is the spread operator (...)?

The spread operator is used to expand elements of an array or properties of an object.

3.What is the difference between map(), filter(), and forEach()?

map() creates a new array by transforming each element.filter() creates a new array based on a condition.forEach() is used to loop through an array and does not return a new array.

4.What is an arrow function?

An arrow function is a shortcut syntax of a JavaScript function. const number=(x,y)=> x+y;

5.What are template literals?

Template literals are a JavaScript feature that allow us to easily write strings and insert variables using $, {}.
